# 개요
매장 예약 서비스 구현

Use : Spring, Jpa, Mysql, Swagger, Jwt, Mailgun

Goal : 사용자와 매장 관리자를 위한 매장 예약 서비스 Backend 구성

참고 : Mailgun SecretKey 노출 위험으로 설정 파일은 공유 안함
